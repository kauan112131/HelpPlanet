# HelpPlanet

## Sobre o Projeto
O **HelpPlanet** é um aplicativo desenvolvido para conscientizar e auxiliar os usuários sobre questões ambientais e sustentabilidade. Nos últimos anos, a preocupação com o meio ambiente e o desenvolvimento sustentável tem se tornado cada vez mais urgente, devido ao aumento da produção de lixo e à degradação dos recursos naturais. Este projeto busca oferecer soluções práticas e educativas para promover hábitos mais sustentáveis.

## Funcionalidades
- Informações e dicas sobre sustentabilidade.
- Monitoramento e gestão de ações ambientais.
- Interface amigável e interativa para os usuários.
- Futuras funcionalidades planejadas para aumentar o engajamento ambiental.

## Tecnologias Utilizadas
- **Linguagem:** Kotlin
- **Plataforma:** Android
- **Build System:** Gradle

## Estrutura do Projeto
- `app/` - Código-fonte principal do aplicativo.
- `gradle/` - Configurações do Gradle.
- `.idea/` - Configurações do projeto no Android Studio.
- Arquivos de build (`build.gradle.kts`, `gradle.properties`) para gerenciamento de dependências e configuração do projeto.
